Bartender Bravo

Animation game

This 1. semester assignment was to design and develop a click-game using HTML, CSS and Javascript, and for me this was also the first time using Javascript. This was an individual task and we had to draw and design each element of the game with matching homemade sound effects. In the end we then had to animate the whole thing making it into a functional click-game. To present the game we had to create a website which enclosed some of the proces creating the game.
